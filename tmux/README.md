# Custom Keymaps
| Actions           | Keys                   |
| ----------------- | ---------------------- |
| prefix            | C-b                    |
| split vertical    | prefix + v             |
| split horizontal  | prefix + h             |
| resize pane       | option(meta) + h,j,k,l |
| toggle status bar | prefix + \             |
